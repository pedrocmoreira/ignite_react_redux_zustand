## 🚀 Instalando o React + Redux + Zustand

Para instalar o projeto, siga estas etapas:

Linux e macOS:
```
git clone https://github.com/pedrocmoreira/ignite_react_redux_zustand.git
cd ignite_react_redux_zustand
npm install
npm run dev
```

Windows:
```
git clone https://github.com/pedrocmoreira/ignite_react_redux_zustand.git
cd ignite_react_redux_zustand
npm install
npm run dev
```
